# Ever House Mobile App Branding Assets

All assets needed for the Ever House mobile app.

## Files

### App Icons
- `icon-512.png` - High-res app icon for App Store
- `icon-192.png` - Standard app icon
- `favicon.ico` - Favicon

### Logos
- `everhouse-logo-light.webp` - Full logo for dark backgrounds
- `everhouse-logo-dark.webp` - Full logo for light backgrounds
- `EH_logo-23.webp` - EH logo (dark version)
- `EH_logo-23_white.webp` - EH logo (white version)

### Monogram (EH)
- `monogram-dark.webp` - EH monogram for light backgrounds
- `monogram-white.webp` - EH monogram for dark backgrounds

### Mascot (Walking Golfer)
- `mascot-dark.webp` - Static mascot for light backgrounds
- `mascot-white.webp` - Static mascot for dark backgrounds
- `EH-guy-icon.webp` - Mascot icon

### Loading Animations
- `walking-mascot-green.gif` - Animated walking golfer (green/dark mode)
- `walking-mascot-white.gif` - Animated walking golfer (white/light mode)

## Usage Notes

- Use the walking GIF animations for loading states
- Use monogram for compact spaces (tab bar, headers)
- Use full logo for splash screens and login
- Match light/dark versions to user's theme preference
